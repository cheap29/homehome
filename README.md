# 自分ほめほめ

うっかりさんや自己否定しやすい人が、日々の小さな成功体験を積み重ねるためのAndroidアプリ。

## アプリ概要

「たくさん管理すること」ではなく「今日やれたことを確認し、自分をほめること」が目的。
- 目標は3つだけ
- 未達成は「持ち越し」（失敗ではない）
- 時間制限なし・生活リズム不規則な人にも対応
- オフライン完結・ログインなし・広告なし

## 技術構成

| 要素 | 採用技術 |
|---|---|
| 言語 | Kotlin |
| UI | Jetpack Compose + Material3 |
| 状態管理 | ViewModel + StateFlow |
| DB | Room |
| 非同期 | Coroutines / Flow |
| ナビゲーション | Navigation Compose |
| DI | Application クラスで手動管理（MVP） |

## 起動方法

1. Android Studio（Hedgehog以降）でプロジェクトを開く
2. `app` モジュールを選択してRunまたはBuild
3. minSdk 26（Android 8.0）以上の実機またはエミュレータで動作確認

## 主要画面

| 画面 | 説明 |
|---|---|
| ホーム | 今日の3つの確認・チェック |
| やることBOX | タスクの追加・編集・削除 |
| 毎日の単語帳 | 行動テンプレートの管理 |
| 今日の3つを選ぶ | 単語帳+BOXから最大3つ選択 |
| 振り返り | 達成確認・ボーナス追加・今日を閉じる |
| 振り返り完了 | ほめコメント表示・スコア確認 |
| 履歴 | 過去の振り返り一覧 |
| 履歴詳細 | その日の目標・チェック結果・ボーナス |

## MVPでできること

- [x] ホーム画面で今日の3つを確認・チェック
- [x] やることBOXにタスク追加・編集・削除
- [x] 単語帳に単語追加・編集・削除
- [x] 今日の3つを選ぶ（最大3つ、やさしく制限）
- [x] 振り返りで「これもやった！」追加
- [x] 振り返り完了でほめコメント表示
- [x] 履歴一覧・詳細確認
- [x] オフライン完結（Firebase・ログイン不要）

## キャラクター画像の差し替え

`app/src/main/res/drawable/` 内の `character_*.xml` を PNG に差し替えることで手描きキャラクターを表示できる。

| ファイル名 | 使用場面 |
|---|---|
| character_home | ホーム画面 |
| character_select_three | 今日の3つ選択 |
| character_limit | 4つ目選択時ダイアログ |
| character_reflection | 振り返り画面 |
| character_bonus | ボーナス追加シート |
| character_complete_great | 振り返り完了（目標達成） |
| character_complete_normal | 振り返り完了（1〜2個） |
| character_complete_bonus | 振り返り完了（目標0・ボーナスあり） |
| character_complete_soft | 振り返り完了（何もなし） |
| character_history | 履歴画面 |
| character_empty | 空状態 |
